# IT Solutions - Sitio Web

Este es el repositorio para el sitio web corporativo de **IT Solutions**, una consultora de tecnología especializada.

## Descripción

El proyecto consiste en una página web moderna, responsive y minimalista, diseñada para presentar los servicios y la identidad de la marca.

## Tecnologías

- **HTML5**: Estructura semántica.
- **CSS3**: Diseño personalizado, Flexbox, Grid, Variables CSS.
- **JavaScript (Vanilla)**: Interactividad y navegación.

## Estructura del Proyecto

```
/
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
├── index.html
└── README.md
```

## Configuración y Uso

1. Clonar el repositorio.
2. Abrir `index.html` en tu navegador web de preferencia.
